WKND is a fictitious adventure and travel website created by Adobe to demonstrate how anyone can use Adobe Experience Manager Helix to build a beautiful, feature-rich website over a single weekend. This site is built entirely with Adobe Experience Manager Core Components and Archetype that are available as open source code to the public. The entire site source code is available as open source as well and is accompanied with a detailed tutorial on how to recreate the site.

Many of the beautiful images in the WKND site are available for purchase via Adobe Stock.
